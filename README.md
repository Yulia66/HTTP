# C++ TCP Raw Socket Server - Пентест инструмент для сканирования

простой TCP-сервер на C++ с использованием сырых сокетов Linux. Предназначен для пентеста и сканирования сетевых уязвимостей.

## Особенности

- Чистые сокеты Linux (без внешних библиотек)
- Корректная обработка ошибок
- HTTP обработчик для веб-сканирования
- Менеджер задач для автоматизации сканирования
- Парсер данных для анализа ответов
- Модели данных для хранения результатов
- Запуск внешних процессов (инструментов пентеста)

## Основные возможности (пентест)

| Функция | Описание |
|---------|----------|
| **Сканирование портов** | Обнаружение открытых портов на целевых хостах |
| **HTTP/HTTPS анализ** | Сбор заголовков, определение серверов |
| **Запуск пентест-тулзов** | Интеграция с Nmap, Hydra, Nikto |
| **Парсинг результатов** | Анализ ответов серверов |
| **Управление задачами** | Очередь и приоритеты сканирования |

## Требования

- Linux OS (Kali Linux, Parrot OS, Ubuntu)
- g++ с поддержкой C++11 или новее
- Стандартные библиотеки POSIX
- Права root (для raw сокетов)

## Установка и запуск

### Компиляция сервера

```bash
# Компиляция объектных файлов
g++ -c server.cpp -o server.o
g++ -c task_manager.cpp -o task_manager.o
g++ -c http_handler.cpp -o http_handler.o
g++ -c parser.cpp -o parser.o
g++ -c models.cpp -o models.o
g++ -c run_process.cpp -o run_process.o

# Линковка
g++ server.o http_handler.o parser.o models.o task_manager.o run_process.o -o server
```


### Компиляция тестового клиента (опционально)
```bash
# Компиляция
g++ -c client.cpp -o client.o

# Линковка
g++ client.o -o client
```
### Запуск клиента

```bash
./client
```

## Использование

1. Запустите сервер
2. Подключитесь через тестовый клиент 
3. Отправляйте текстовые сообщения
4. Сервер преобразует их в ВЕРХНИЙ РЕГИСТР и отправит обратно
